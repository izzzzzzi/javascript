# Lexical grammar

1. control character 제어문자
2. white space 공백문자
3. line terminators 개행문자
4. comments 코멘트
5. keyword 예약어
6. literals 리터럴

# Language element

## statements 문

1. 공문, 식문, 제어문, 선언문
2. 단문, 중문

## expression 식

- 하나의 값으로 귀결됌
- 식은 값이다.
  - 값은 저장하지 않으면 메모리에 휘발되어 없어짐

1. 값식, 연산식, 호출식

## identifier 식별자

- 변수는 메모리 주소의 별명
-

1. 기본형, 참조형
2. 변수, 상수
